### learn-react-context-and-hooks

---
#### 下载项目文件
```
git clone https://github.com/sincerity628/learn-react-context-and-hooks.git
```
#### 安装依赖
```
cd learn-react-context-and-hooks
npm install
```
#### 运行项目
```
npm start
```
